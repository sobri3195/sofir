feat: Add full Bit Integration plugin compatibility with 10 triggers and 3 actions

Add comprehensive Bit Integration support to enable workflow automation between
SOFIR and 200+ external applications including Mailchimp, Slack, Google Sheets,
HubSpot, and more.

## What's Added

### Core Integration (669 lines)
- New BitIntegration class with singleton pattern
- Auto-loaded via SOFIR Loader
- Zero performance impact when Bit Integration not installed
- Priority 999 for all handlers (runs after core processing)

### 10 Comprehensive Triggers
1. User Registered - with phone, name, email, roles support
2. User Profile Updated - with all profile fields
3. User Logged In - with timestamp tracking
4. Payment Completed - supports all gateways (Manual/Duitku/Xendit/Midtrans)
5. Form Submitted - with full form data
6. Post Published - with permalink and metadata
7. Comment Posted - with author and content
8. Membership Changed - with old/new plan tracking
9. Appointment Created - with datetime, provider, client
10. Appointment Updated - with status change tracking

### 3 Powerful Actions
1. Create User - accepts phone number, supports SOFIR phone-only registration
2. Update User - modify existing user profiles from external apps
3. Create Post - create any post type from external sources

### Brand Assets
- SOFIR icon (SVG) with purple gradient brand colors
- Displays in Bit Integration UI

## Integration Points

### With Existing Modules
- Payments: Hooks into `sofir/payment/status_changed`
- Appointments: Hooks into appointment created/updated events
- Membership: Ready for membership change events
- WordPress Core: user_register, profile_update, wp_login, publish_post, comment_post

### Bit Integration Hooks
- Registers via `btcbi_trigger` filter
- Registers via `btcbi_action` filter
- Executes via `btcbi_trigger_execute` action
