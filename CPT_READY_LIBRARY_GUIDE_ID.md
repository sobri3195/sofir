## ğŸ’¡ Cara Menggunakan

### Step 1: Akses Library Tab
1. Login ke WordPress Admin
2. Klik menu **SOFIR** di sidebar
3. Pilih tab **Library**
4. Scroll ke section **ğŸ Ready-to-Use CPT Library**

### Step 2: Pilih Template
Lihat 5 template cards yang tersedia. Setiap card menampilkan:
- **Icon & Badge**: Visual identifier dan status (Popular/New/Simple/Pro)
- **Nama Template**: Jenis website yang bisa dibuat
- **Deskripsi**: Penjelasan singkat fitur utama
- **Features List**: Daftar lengkap fitur yang included

### Step 3: Install Template
1. Klik tombol **+ Install Template** pada card yang dipilih
2. Sistem akan otomatis:
   - Mendaftarkan Custom Post Type
   - Setup semua custom fields
   - Mendaftarkan taxonomies
   - Mengaktifkan filters
   - Flush rewrite rules
3. Halaman akan refresh dengan success message
4. Template sudah siap digunakan!

### Step 4: Verifikasi Instalasi
1. Cek menu WordPress Admin - CPT baru akan muncul di sidebar
2. Klik menu CPT untuk mulai menambahkan konten
3. Test di frontend untuk melihat hasilnya

### Step 5: Refresh Permalink (Penting!)
1. Pergi ke **Settings â†’ Permalinks**
2. Klik **Save Changes** (tanpa ubah apapun)
3. Ini memastikan rewrite rules berfungsi dengan benar

## ğŸ› Troubleshooting

### Template Tidak Muncul di Menu
**Problem**: Setelah install, CPT tidak muncul di menu admin

**Solution**:
1. Hard refresh browser (Ctrl+F5)
2. Logout dan login kembali
3. Check user capabilities (harus `manage_options`)

### 404 Error di Single/Archive Page
**Problem**: Halaman CPT menampilkan 404 Not Found

**Solution**:
1. Pergi ke **Settings â†’ Permalinks**
2. Klik **Save Changes**
3. Refresh rewrite rules
4. Clear browser cache

### Fields Tidak Muncul di Edit Post
**Problem**: Custom fields tidak tampil di post editor

**Solution**:
1. Check Screen Options (pojok kanan atas)
2. Pastikan custom fields di-check
3. Reload halaman
4. Clear WordPress object cache

### Import Gagal
**Problem**: Error saat import JSON file

**Solution**:
1. Validate JSON syntax (gunakan jsonlint.com)
2. Check file size limit di php.ini
3. Check file permissions
4. Check error log WordPress

### Template Sudah Terinstall Tapi Ingin Re-install
**Problem**: Ingin install ulang template

**Solution**:
1. Pergi ke SOFIR â†’ Content
2. Delete CPT yang existing
3. Kembali ke Library tab
4. Install ulang template
