## ğŸ”§ Troubleshooting

### Pattern tidak muncul setelah paste
**Fix:** Pastikan Anda di Block Editor (Gutenberg), bukan Classic Editor

### Style tidak sesuai
**Fix:** Theme Anda mungkin override styles. Tambah custom CSS di Appearance â†’ Customize

### Navigation menu kosong
**Fix:** Buat menu dulu di Appearance â†’ Menus, lalu assign ke navigation block

### Social icons tidak muncul
**Fix:** Isi URL di setiap social link, atau hapus yang tidak digunakan

### Copy button tidak berfungsi
**Fix:** Browser mungkin block clipboard access. Use manual copy dari modal.
